# 📚 Library Management System

A Library Management System built using **Java**, **Spring Boot**, **Spring Data JPA**, **Thymeleaf**, **MySQL**, and **Spring Security**.

This project is being developed to demonstrate enterprise-level backend development practices using the Spring ecosystem.

---

## 🛠️ Getting Started

### Clone Repository

```bash
git clone https://github.com/sannal123/library-management-system.git
```

### Navigate to Project

```bash
cd library-management-system
```

### Configure Database

Create a MySQL database:

```sql
CREATE DATABASE library_db;
```

Update `application.properties` with your MySQL username and password.

### Run Application

```bash
mvn spring-boot:run
```

Open:

```
http://localhost:8080
```
